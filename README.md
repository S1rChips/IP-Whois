## ✨ ویژگی‌ها

- 🌐 **جستجوی جامع IP** - دریافت اطلاعات کامل جغرافیایی، ISP و سازمان
- 🏢 **تشخیص دیتاسنتر** - شناسایی IPهای متعلق به دیتاسنترها و سرویس‌های ابری
- 🔗 **بررسی دامنه** - دریافت اطلاعات Whois، تاریخ ثبت و انقضا
- 🎯 **تشخیص خودکار** - تشخیص هوشمند نوع ورودی (IP یا Domain)
- 🚀 **سریع و کارآمد** - استفاده از APIهای مختلف برای دریافت اطلاعات دقیق
